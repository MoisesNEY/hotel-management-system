# LIBRERIAS USASADAS
1. react ^19.2.0
2. react-dom ^19.2.0
3. lucide-react ^0.560.0 (iconos)
4. next-themes (cambio de tema)
5. react-router-dom
6. styled-components

# DEPENDENCIAS DE DESAROLLO
1. vite ^7.2.4 (bundler y servidor de desarrollo)
2. @vitejs/plugin-react ^5.1.1 (plugin de React para Vite)
3. tailwindcss ^4.1.18 (framework CSS)
4. postcss ^8.5.6 (procesador CSS)
5. autoprefixer ^10.4.22 (agrega prefijos CSS automáticamente)
6. typescript ~5.9.3
7. @types/react, @types/react-dom, @types/node (tipos para TypeScript)
8. eslint y plugins relacionados (linting y validación de código)


# TECNOLGIAS USADAS
1. React + TypeScript: Desarrollo de componentes y páginas.
2. Vite: Bundler rápido y servidor de desarrollo.
3. TailwindCSS v4: Framework CSS para estilos utilitarios.
4. PostCSS + Autoprefixer: Procesamiento y compatibilidad de CSS.
5. ESLint: Validación de código y buenas prácticas.
